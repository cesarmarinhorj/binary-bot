
# Onebotmembers

Robo de operações binárias.

## Como rodar

Antes é preciso instalar a versão 10.15.3 do node:

```
nvm install 10.15.3
nvm use 10.15.3
```

```
git clone https://github.com/mbmaciel/onebotmembers.git
cd onebotmembers
npm install
npm start
```


## Atualizar para ultima versão

```
git pull 
npm install
```
